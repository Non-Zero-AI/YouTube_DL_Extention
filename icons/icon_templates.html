YouTube Downloader Pro - Icon Templates

Instructions

These are SVG templates for the extension icons. You can:

  1. Right-click on each icon and select "Save image as..." to save them as SVG files
  2. Take screenshots of each icon and save them as PNG files
  3. Copy the SVG code and save it to the appropriate files

The icons should be saved as:

  • icon16.png (16x16 pixels)
  • icon48.png (48x48 pixels)
  • icon128.png (128x128 pixels)
